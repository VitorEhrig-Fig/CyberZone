Sobre

O site é uma iniciativa de um projeto para o mundo de TI, onde vai englobar tutoriais, artigos, referencias e etc de assuntos de ti em geral como programação, hardware, criptomoedas, etc. Haverá um fórum para debates de dúvidas. Havera tela de login para usuario padrao, onde, o msm podera criar um artigo e enviar quase q um pull request para os administradores q, em suma, irão ler este artigo e postar no site. Cada artigo postado tera o nome do usuario q postou. Nao somente artigos mas tambem questoes em foruns ou tutoriais para as pessoas poderão ser postados. A questao de perfis sera amplamente focada, ou seja, cada usuario podera customizar o seu proprio perfil e havera um sistema de ranking tbm.
